5
N 네이버 열기 → 붙여넣기 → 이미지 업로드 → 발행
🖼️ 이미지 생성 안내
이미지는 Gemini API를 통한 나노바나나 (gemini-3.1-flash-image-preview)로 생성됩니다. 같은 API 키로 텍스트 + 이미지 모두 생성! Claude 사용 시 Pollinations AI로 자동 전환됩니다.
❓ 네이버 자동 업로드가 안 되는 이유
네이버 블로그 글쓰기 API는 2023년 완전 종료됐습니다. 복사-붙여넣기 방식이 가장 안전합니다.
👥 지인과 공유하기
내 API 키가 내장된 공유 전용 파일을 생성합니다.
지인은 API 키 없이 바로 사용 가능합니다.
⚠️ 생성된 파일에는 API 키가 포함되니 신뢰하는 분에게만 공유하세요.
